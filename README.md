# Hackcamp Group 21: TilePlanUK

## Project Scope
> Soon-to-be Modern survey solution intended to be used for the TilePlanUK company to replace the current error-prone solution. Will likely utilize **PHP 8** and **Javascript.**


## Overall file-structure:
> When commiting code to this repository please adhere to the MVC structure as it ensures everything is organized. All standard PHP files that are intended to be used as *CONTROLLERS* are meant to be stored in the root folder whereas Model files are contained in its own folder. *MODELS* are stored in the /Model folder and *VIEWS* will be stored in the /Views channel.
